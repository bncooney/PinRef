# PinRef - Visual Reference Image Viewer

Keep your reference images always visible while you work. PinRef is a lightweight desktop tool that lets you drag and drop multiple images onto a customizable canvas. Perfect for artists, designers, and developers who need to reference visual materials while working in other applications.

## Features:

- Pin window on top of all other apps for constant visibility
- Drag and drop multiple images onto a zoomable canvas
- Move, resize, and arrange images freely
- Auto-reload images when files change (great for iterative work)
- Zoom and pan with mouse wheel and drag
- Clean, minimal interface that stays out of your way

## System Requirements

- Windows 10 (version 14393 or later) or Windows 11
- .NET 10.0 Runtime

## Installation

1. Get the latest release from the Microsoft Store
2. Launch PinRef from your Start Menu

## User Guide

### Getting Started

1. **Launch the application** - Find PinRef in your Start Menu or pin it to your taskbar for quick access

2. **Add images to your canvas**
   - Drag and drop image files directly onto the canvas
   - Supports common formats: PNG, JPG, JPEG, BMP, GIF
   - You can add multiple images at once

3. **Keep the window on top**
   - When pinned, the window stays on top of all other applications
   - This allows you to reference your images while working in other programs

### Working with Images

#### Moving Images
- **Click and drag** any image to reposition it on the canvas
- Images can be placed anywhere on the canvas, even overlapping

#### Resizing Images
- **Click and drag the corners** or edges of any image to resize
- Hold **Shift** while resizing to maintain aspect ratio
- Each image can be sized independently

#### Removing Images
- **Right-click** on an image and select "Remove" from the context menu
- Or select the image and press **Delete** key

### Canvas Controls

#### Zooming
- **Mouse wheel up/down** - Zoom in/out on the canvas
- Zoom is centered on your mouse cursor position
- All images scale together with the canvas zoom level

#### Panning
- **Middle mouse button + drag** on empty canvas space to pan around
- Useful when zoomed in to navigate to different areas

### Auto-Reload Feature

PinRef automatically watches your image files for changes:
- When you save a new version of an image file, it updates automatically on the canvas
- Perfect for iterative workflows (e.g., updating concept art, UI mockups)
- Position and size are preserved when images reload
- No need to manually refresh or re-add images
